
# Caixa360

## Descrição

Caixa360 é uma funcionalidade que permite gerenciar fluxos de caixa através de:

### Funcionalidades Principais

- **Entrada de Dinheiro**: Registre e controle todas as receitas e depósitos
- **Saída de Dinheiro**: Acompanhe despesas, saques e pagamentos realizados
- **Consulta de Extrato**: Visualize o histórico completo de transações com detalhes de data, valor e tipo

### Características

- Interface intuitiva para registro rápido de transações
- Filtros avançados no extrato por período, tipo ou valor
- Saldo atualizado em tempo real
- Relatórios detalhados das movimentações


## Tecnologia

### Streamlit

Caixa360 é desenvolvido com **Streamlit**, um framework Python moderno que permite criar aplicações web interativas de forma rápida e eficiente. Streamlit oferece:

- **Desenvolvimento multiplataforma**: Funciona perfeitamente em navegadores desktop e mobile
- **Interface responsiva**: Adapta-se automaticamente a diferentes tamanhos de tela
- **Prototipagem rápida**: Sem necessidade de HTML, CSS ou JavaScript complexo
- **Deployment simples**: Deploy em poucos cliques na Streamlit Cloud ou servidores próprios
- **Componentes prontos**: Botões, gráficos, tabelas e filtros já otimizados para web
